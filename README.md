# AuditChain

AuditChain is a decentralized invoice auditing system that combines **Deep Learning (CNN)** for risk assessment and **Blockchain** for immutable record-keeping.

## Features

- **AI-Powered Audit:** Automatically evaluates invoice reliability using a trained CNN model.
- **Blockchain Integration:** Secures all certified invoices on a tamper-proof ledger.
- **Dual Interface:** Dedicated portals for Accountants (Comptable) and Auditors.

## Tech Stack

- **Frontend/Backend:** Streamlit (Python)
- **AI Model:** TensorFlow/Keras (CNN)
- **Blockchain:** Firefly / Hyperledger
